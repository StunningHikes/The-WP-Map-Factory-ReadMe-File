The easy way to create great looking Google maps for your Wordpress website.

== Description ==

The Map Factory Wordpress plugin allows any number of maps to be easily created. Add placecards, any number of markers, circles, polylines and polygons and even elevation charts.

To display maps with this plugin you will need a Google Maps JavaScript API key - this can be easily obtained from Google.

It was never so easy to create maps in Wordpress!

== Installation ==

You will have received a copy of the plugin as a zip file. You can  upload it and install it through the Plugins admin screen.
1. Navigate to Plugins > Add New.
2. Click the Upload Plugin button at the top of the screen.
3. Select the zip file from your local filesystem.
4. Click the Install Now button.
5. When installation is complete, you’ll see “Plugin installed successfully.” Click the Activate Plugin button at the bottom of the page.
